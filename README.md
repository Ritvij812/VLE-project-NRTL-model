# Vapor–Liquid Equilibrium Modeling using NRTL Model

## Overview

This project performs Vapor–Liquid Equilibrium (VLE) analysis for non-ideal binary mixtures using the NRTL thermodynamic model in LibreOffice Calc.

## Tools Used

* LibreOffice Calc
* NRTL Model
* Spreadsheet Solver

## Features

* Activity coefficient calculation
* Phase equilibrium analysis
* Solver-based parameter estimation
* Data visualization using plots and tables

## Learning Outcomes

* Understanding of VLE behavior
* Thermodynamic modeling
* Numerical analysis using spreadsheets
